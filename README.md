# auto_data_collector
auto data collector

## Setup

### Virtual Environment

This project uses a Python virtual environment located at `.venv/` (Python 3.13.0).

**Create the virtual environment:**
```bash
python3 -m venv .venv
```

**Activate the virtual environment:**
```bash
source .venv/bin/activate
```

**Deactivate when done:**
```bash
deactivate
```
